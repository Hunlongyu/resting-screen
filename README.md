### ✨ 特性
1. 全局快捷键 ALT + L
2. 支持锁屏 + 息屏
3. 支持开机自启
4. 无界面，托盘图标
5. 软件：78KB， 安装包：1.5MB，占内存：6.5MB
6. 支持 x86，x64 系统


### 👀 说明

* 息屏：让显示器进入待机模式节省电力，不影响后台进程。唤醒时继续之前操作，无需密码（除非已设定）。
* 锁屏：显示登录界面，需密码才能访问系统，确保无授权用户无法使用，适用于临时离开电脑。后台任务继续运行。

总的来说，息屏主要用于节省电力，而锁屏主要用于提升账户安全。
