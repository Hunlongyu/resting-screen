## 息屏助手

启动后电脑立即息屏，没有窗口，常驻任务栏，双击托盘图标，出现半透明窗口，双击窗口立即息屏
也可以使用全局快捷键 ALT + L

* 息屏：屏幕变黑，相当于关闭屏幕电源。
* 锁屏：屏幕被锁住，但不会变黑，如果有密码的话，进入桌面还需要输入密码。

### 软件配置
小软件就没有写设置界面了。在软件同级目录下有个 config.ini 文件
```ini
[Application]
; 软件启动时，自动熄屏
start=true
; 软件启动后，不显示窗口
window=false
; Alt + L 快捷键是否生效 
shortcut=true
```
也很简单，自行修改，不会修改的就不要动了，或者先做个备份。
